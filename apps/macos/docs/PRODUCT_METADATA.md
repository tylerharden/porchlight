## Description

Porchlight helps developers see and manage the local servers running on their Mac.

It lives in the menu bar, scans local listening ports, and shows active and recently seen development servers with quick actions to open, start, stop, pin, remove, and organize them into groups.

Porchlight is designed for local development workflows. It uses a bundled Rust CLI as the scanning engine and a native macOS companion app for menu bar and settings UI.

Key features:

- See active and recent local server ports from the menu bar.
- Open localhost URLs quickly.
- Stop active development servers from Porchlight.
- Restart recently seen servers from saved start commands.
- Pin important servers so they stay visible.
- Organize servers into configurable groups.
- Auto-detect common project favicons for project identity.
- Tune refresh behavior and menu bar visibility.
- Optional launch at login.

Porchlight runs locally and does not send local process, port, command, or project data to Tyler Harden or third-party analytics services.
