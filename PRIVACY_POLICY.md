# Porchlight Privacy Policy

Last updated: July 22, 2026

Porchlight is designed to run locally on your Mac.

## Information Processed Locally

Porchlight inspects local listening ports and process metadata so it can show development servers. This may include:

- Port numbers
- Process names and process identifiers
- Commands used to start local servers
- Working directories
- Server status
- Pinned and recent server state
- Group, colour, and icon settings

This information is processed locally on your device and may be stored locally using app preferences and Porchlight configuration/state files.

## Information Sent Off Device

Porchlight does not create an account, track usage, include third-party analytics, or send your local server/process data to Tyler Harden or analytics services.

If you choose to open GitHub links, report issues, download releases, or interact with the project repository, GitHub may process information according to GitHub's own terms and privacy policy.

## Local Data Removal

You can remove Porchlight's local state by deleting its app preferences and files under:

- `~/.config/porchlight`
- `~/.local/state/porchlight`

## Other Platforms

This policy covers Porchlight for macOS. The same product-wide privacy approach should also apply to the Porchlight CLI and future platform builds as long as they continue to process data locally and do not transmit server/process data. If a future platform adds different data collection or network services, this policy should be updated with a platform-specific section.
